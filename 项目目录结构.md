## 项目结构

```
catpool_sms_client/
├── app.py                      # 主程序入口
├── project_check.py            # 检查程序文件完整性
├── README.md                   # 项目说明文档
├── requirements.txt            # 依赖库
├── .env                        # 环境变量配置文件
├── .gitignore                  # Git忽略文件
├── config/                     # 配置相关
│   ├── __init__.py
│   ├── settings.py             # 基础设置管理
│   └── logging_config.py       # 日志配置
├── database/                   # 数据库相关
│   ├── __init__.py
│   └── connection.py           # 数据库连接管理
├── models/                     # 数据模型层
│   ├── __init__.py
│   ├── base.py                 # 基础模型类
│   ├── user.py                 # 用户模型（仅操作用户）
│   ├── task.py                 # 任务模型
│   ├── message.py              # 消息明细模型
│   └── port.py                 # 端口设备模型
├── services/                   # 业务逻辑层
│   ├── __init__.py
│   ├── auth_service.py         # 认证服务（登录验证、MAC验证）
│   ├── task_service.py         # 任务管理服务（创建、执行、监控）
│   ├── port_service.py         # 端口管理服务（扫描、状态监控）
│   ├── message_service.py      # 消息处理服务（发送、监测提醒）
│   └── credit_service.py       # 积分管理服务（查询、扣除、定时更新）
├── core/                       # 核心功能
│   ├── __init__.py
│   ├── port_scanner.py         # 端口扫描器（COM端口检测）
│   ├── task_executor.py        # 任务执行器（任务调度）
│   ├── message_sender.py       # 短信发送器（串口通信）
│   ├── monitor_detector.py     # 监测检测器（TX条检测提醒）
│   ├── file_handler.py         # 文件处理（号码导入/导出）
│   └── utils.py                # 常用工具函数
├── ui/                         # 用户界面
│   ├── __init__.py
│   ├── login_window.py         # 登录窗口
│   ├── main_window.py          # 主窗口（三区域布局）
│   ├── components/             # 界面组件
│   │   ├── __init__.py
│   │   ├── task_list_widget.py # 任务列表组件（左侧）
│   │   ├── port_grid_widget.py # 端口网格组件（右侧）
│   │   ├── status_bar.py       # 状态栏组件（顶部）
│   │   └── timer_widget.py     # 定时器组件（五分钟更新）
│   ├── dialogs/                # 对话框
│   │   ├── __init__.py
│   │   ├── add_task_dialog.py  # 添加任务对话框
│   │   ├── task_test_dialog.py # 任务测试对话框
│   │   ├── task_edit_dialog.py # 修改内容对话框
│   │   ├── config_dialog.py    # 选项配置对话框
│   │   └── export_dialog.py    # 导出对话框
│   └── styles/                 # 样式文件
│       ├── __init__.py
│       └── orange_theme.qss    # 橙色主题样式
├── static/                     # 静态资源
│   └── icons/                  # 图标文件
│       ├── logo.png            # 应用图标
│       ├── start.png           # 开始图标
│       ├── stop.png            # 停止图标
│       ├── pause.png           # 暂停图标
│       ├── add.png             # 添加图标
│       ├── edit.png            # 编辑图标
│       ├── delete.png          # 删除图标
│       ├── export.png          # 导出图标
│       ├── test.png            # 测试图标
│       ├── retry.png           # 重试图标
│       └── settings.png        # 设置图标
├── temp/                       # 临时文件
│   ├── uploads/                # 上传的号码文件
│   ├── exports/                # 导出的结果文件
│   └── logs/                   # 应用日志
│       ├── app.log             # 应用日志
│       ├── task.log            # 任务日志
│       ├── port.log            # 端口日志
│       └── error.log           # 错误日志
└── tests/                      # 测试文件
    ├── __init__.py
    └── test_basic.py           # 基础功能测试
